# karnoweb/livewire-datepicker

Jalali/Gregorian datepicker for Livewire 3 with Alpine.js.

## Installation

```bash
composer require karnoweb/livewire-datepicker
```

## Usage

```blade
<x-datepicker wire:model="birth_date" jalali export-calendar="gregorian" export-format="Y-m-d" />
```

## Namespace

- **Package:** `karnoweb/livewire-datepicker`
- **PHP namespace:** `Karnoweb\LivewireDatepicker`
